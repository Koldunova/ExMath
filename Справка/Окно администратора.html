Окно администратора

После авторизации как «Администратор» будет доступно:

 1. Информация о зарегистрированных учащихся (прогресс);

2. Добавление и редактирование тестирующего и теоретического материалов. Так же в этом режиме можно передать информацию для учащихся, ознакомится со «Справкой» и подключить базу данных.

Подключить базы данных 

Данная кнопка необходима для подключения базы данных (которая находится на сервере).  Данная настройка необходима на каждом ПК.  Откроется диалоговое окно, где надо указать путь к базе данных. 

Редактировать материалы

Мы можете изменять теоретическую часть (допустим если перепутали страницы, либо хотите сменить картинку), в тестах изменить букву правильного ответа

Добавить тест/учебник

Для добавления материалов были реализованы форма-юнит в которой можно загрузить учебный и тестирующий материал.

Добавление тестирующего материала:

 1. Название теста;

2. Количество вопросов;

3. Время на тестирование (в секундах);

4. Задания в картинках;

5. Правильный ответ (пустые поля нельзя оставлять!);

6.После заполнения нажать кнопку добавить --> сохранить. 

Добавление учебного материала:

 1. Название раздела;

2. Количество глав;

3. Название главы;

4. Текст в картинках (скриншот, Photoshop и т.д.);

6.После заполнения нажать кнопку добавить --> сохранить.
